Recorrido de 10 Ejercicios para Aprender JavaScript

1. Variables y Tipos — Guardar información

Declara variables y muestra sus valores por consola.

2. console.log() — Ver qué pasa

Imprime en la consola una frase con tu nombre y edad.

3. Operadores — Trabajar con los datos

Operaciones básicas entre dos números y muestra resultados en consola.

4. Condicionales — Tomar decisiones

Verifica si un número es par o impar.

5. Bucles — Repetir tareas

Muestra los números del 1 al 5 en consola usando un bucle.

6. Arrays — Agrupar datos

Muestra una fruta específica desde un array.

7. Bucles con arrays — Automatizar recorridos

Recorre y muestra todos los elementos de un array.

8. Funciones — Reutilizar código

Crear y usar una función para saludar.

9. Arrays de objetos — Datos más reales

Imprime el nombre de la persona más joven desde un array de objetos.
